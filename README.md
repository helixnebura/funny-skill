# 馕言文 Skill

一个把普通中文改写成「馕言文」的 Codex Skill。

它不是普通的新疆口语翻译器，而是偏互联网梗式的风格转换：保留原意，用倒装、直译感、奇特比喻和一点新疆味口头词，把句子改成热乎、离谱、但还能看懂的表达。

## 效果示例

普通话：

```text
你这个相机拍照真清晰
```

馕言文：

```text
哎捧油，你这个相机拍照雪山泉水一样的亮眼有的呢！
```

普通话：

```text
你家里这个鹦鹉真听话
```

馕言文：

```text
哎捧油，你家里这个鹦鹉巴掌大的小脑袋装着绵羊一样的安静有的呢。
```

普通话：

```text
我不想写作业
```

馕言文：

```text
哎捧油，写作业这事整个人馕坑里闷久了的面团蔫巴有的呢。
```

## 风格特点

- 常用 `哎捧油`、`嘛`、`嘞`、`有的呢` 等口头表达。
- 尽量不用普通话里的 `像`，更偏 `X一样的Y有的呢` 这种别扭直译结构。
- 比喻要贴着主体，不要先解释再比喻。
- 把 `第一`、`很棒`、`不想`、`不会` 这类判断词压成画面。
- 输出要短，一击就收，不拖解释尾巴。

## 安装

把 `nangyanwen` 文件夹放到 Codex skills 目录，例如：

```text
~/.codex/skills/nangyanwen
```

Windows 上通常可以放到：

```text
C:\Users\<你的用户名>\.codex\skills\nangyanwen
```

## 使用

在 Codex 里调用：

```text
Use $nangyanwen to convert: 这个药一点也不苦
```

示例输出：

```text
哎捧油，这个药蜜糖馕一样的清甜有的呢。
```
